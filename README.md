# Neha & Prajwal — Wedding Invitation

An animated single-page digital wedding invitation (book format).

- **Couple:** Neha & Prajwal
- **Date:** Saturday, 27 June 2026 (Ashar 13, 2083 B.S.), 12:00 PM – 2:00 PM
- **Venue:** Yala Durbar, Sankhamul, Lalitpur

## Files
- `index.html` — main page (HTML + CSS, loads `book.jsx`)
- `book.jsx` — all React components (cover, pages, countdown, navigation)
- `assets/couple.jpg` — couple photo

## Run locally
Open `index.html` in a browser. (It loads React via CDN, so an internet connection is needed.)

## Publish (GitHub Pages)
Push to a repo, then **Settings → Pages → Build from branch → `main` / root**.
Your invite goes live at `https://<username>.github.io/<repo>/`.
